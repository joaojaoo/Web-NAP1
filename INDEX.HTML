Este projeto consiste no desenvolvimento de uma Landing Page completa, aplicando os conceitos fundamentais de desenvolvimento web front-end. A página contempla estrutura semântica em HTML, estilização avançada com CSS, responsividade para diferentes dispositivos, criação de tabelas para organização de dados, formulário interativo com validação em JavaScript e navegação por hiperlinks internos.

O trabalho foi desenvolvido como parte da avaliação NAP 01, demonstrando a capacidade de integrar diferentes tecnologias web em um único projeto funcional e bem estruturado.
